# Home

## ICESEE: Ice Sheet State Estimator

Welcome to the **ICESEE GHUB Tool** — an interactive **Jupyter Book** interface for exploring and running the *Ice Sheet State Estimator (ICESEE)* framework.

ICESEE is an open-source framework for **ice-sheet state estimation** and **parameter inference** using ensemble-based data assimilation. Its core goal is to make transient Ensemble Kalman filtering workflows easier to apply across a range of ice-sheet and idealized models by connecting observations (glaciological and geological) with physics-based simulations.

### What this GHUB Tool provides

This GHUB deployment is designed to make ICESEE accessible through a lightweight, browser-based environment.

It provides:

- **Clear documentation** of ICESEE concepts and workflow  
- **Portable, self-contained examples** that run directly in GHUB  
- A guided path from **idealized systems → ice-sheet applications**  
- A stable wrapper around the upstream ICESEE repository
